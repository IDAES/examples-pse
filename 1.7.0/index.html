IDAES ExamplesÂ¶

These documentation pages contain examples of building and optimizing models with the IDAES PSE toolkit, as well as using applications that extend or augment the core IDAES functionality.

  • IDAES Workshop Materials provide examples from workshops for building, optimizing, and analyzing models

  • IDAES Tutorials provide tutorials for building, optimizing, and analyzing models

  • Surrogate Modeling Examples demonstrate the surrogate modeling capabilities of PySMO, ALAMO, and RIPE.

  • MatOpt Examples Show how to use the MatOpt application

Note

All the examples here are available for installation as part of the IDAES-PSE software. Please see the IDAES PSE documentation for instructions on installing and running the examples on your own computer.
